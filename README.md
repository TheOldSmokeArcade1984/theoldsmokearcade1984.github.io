## >_ COS'È QUESTO PROGETTO?
Non è un semplice catalogo o una landing page, ma un vero e proprio **Sistema Operativo Web (PWA)**. Un hub immersivo progettato per racchiudere e lanciare la libreria di videogiochi indipendenti firmata *TheOldSmokeArcade1984*. 

Radicato in un'estetica **retro-arcade**, **cyberpunk** e **synthwave**, l'OS trasforma la navigazione stessa in un'esperienza interattiva. Inserisci la moneta nel cabinato virtuale e sblocca l'accesso al terminale.

## ⚡ FEATURES PRINCIPALI
* **🎮 In-App Gaming Engine:** I giochi si avviano all'interno di un iframe dedicato a tutto schermo. Puoi giocare e tornare alla dashboard di sistema con un solo tocco, senza mai uscire dall'ecosistema dell'hub.
* **📱 Progressive Web App (PWA) Reattiva:** Progettata "Mobile-First" e installabile nativamente su smartphone e PC. Il CSS dinamico supporta fluidamente la rotazione del dispositivo, adattando i controlli touch sia per la modalità *Portrait* che *Landscape* su schermi ultrawide.
* **💽 Estetica CRT & Neon:** Interfaccia hacker-style con filtri scanline globali, bagliori dinamici e due sezioni tematiche principali: `Laser Zone` (Rosa/Ciano) e `Plasma Zone` (Viola/Giallo).
* **🔊 Audio Engine Integrato:** Nessun semplice file MP3 riprodotto in loop. L'hub utilizza le Web Audio API per generare un sintetizzatore in tempo reale: drone sonoro di sottofondo, feedback tattili sui pulsanti e una sequenza acustica di boot procedurale.
* **🛡️ Privacy by Design:** Architettura pulita. Nessun tracciamento, niente analytics invisibili o server cloud. Tutti i dati e i salvataggi dei giochi vengono gestiti e mantenuti strettamente in locale.

## 📂 STRUTTURA DEI NODI
Il sistema smista la libreria su due canali di accesso:
1. **/// Server Hub [GitHub]:** La directory primaria per il web-gaming immediato. Include titoli completi come *The 12 Arcade Collection*, l'infiltrazione tattica di *Octacore: Null Sector*, l'azione frenetica di *Neon Specter* e la logica di *Knight's Tour*.
2. **>_ Mirror Nodes [itch.io]:** Il gateway di connessione remota per scaricare i pacchetti eseguibili standalone o supportare lo sviluppo sulla piattaforma itch.io.

## 🛠️ INIZIALIZZAZIONE E INSTALLAZIONE
1. Raggiungi l'URL principale su `https://theoldsmokearcade1984.github.io/`.
2. Segui le istruzioni a schermo: **Trascina la moneta** verso la fessura per innescare l'avvio.
3. Dal menu di navigazione superiore, usa il tasto `[ APP ]` per forzare la creazione del WebAPK e installare l'hub direttamente sulla schermata Home del tuo dispositivo come un'applicazione nativa.
4. Premi `[ 🔊 ]` per attivare l'esperienza acustica completa.
